# 移动端豆瓣电影

## 技术栈:

Jquery、ES6、ajax、HTML5、CSS3、webpack、less


## 描述:

这是一个移动端豆瓣电影，调用豆瓣api接口.

利用jsonp跨域展示豆瓣TOP250排行榜、北美排行榜、搜索功能。

运用**面向对象封装函数**，用**webpack对代码进行打包优化页面**,主要用jquery编写，css方面使用less，字体图标等。
